# Проект по использованию python, selenium, pytest

## Этапы
```
1. Открыть папку с проектом в gitbash
2. Проверить версию пайтон ```(py --version)```
3. Создать папку venv (python"version python" -m venv "name")
4. Скачать .gitignore(github)
5. Добавить в .gitignore "name"(чтобы не трогать папку)
6. Создать вертуальное окружение(source "name"/Scripts/activate)
7. Установка selenium(pip install selenium)
8. Установка requirements.txt(pip freezer > requirements.txt)
```
